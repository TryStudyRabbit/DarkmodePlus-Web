Getting Started with DarkMode+

DarkMode+ is a powerful Chrome extension that transforms any website with customizable dark mode styling. This guide will help you get started and make the most of all its features.

Note

DarkMode+ works with Chrome version 88 and above. For optimal performance, we recommend keeping your browser updated to the latest version.

Key features of DarkMode+ include:

Universal Dark Mode

Works on virtually any website to create a comfortable dark appearance.

Multiple Themes

Choose from Default, Sepia, Midnight, or custom color schemes.

Site Exclusions

Exclude specific websites from dark mode application.

Color Customization

Fine-tune background, text, links, and accent colors.

Installation

Installing DarkMode+ is quick and easy:

  1. Visit the Chrome Web Store
  2. Search for "DarkMode+" or use our direct link from the homepage
  3. Click the "Add to Chrome" button
  4. Confirm the installation when prompted
  5. The DarkMode+ icon will appear in your browser toolbar
Tip

You can pin DarkMode+ to your toolbar for easier access by right-clicking the icon and selecting "Pin".

After installation, the extension will be active with default settings. You can immediately start using it or customize the settings to your preferences.

Basic Usage

Using DarkMode+ is straightforward:

Enabling/Disabling Dark Mode

Click the DarkMode+ icon in your browser toolbar to toggle dark mode on or off for the current website. When enabled, a checkmark appears on the icon.

Accessing Options

To access the full settings panel:

  1. Click the DarkMode+ icon in your browser toolbar
  2. Click the gear icon ⚙️ in the popup menu
  3. Alternatively, right-click the extension icon and select "Options"

Quick Toggle

You can quickly toggle dark mode on/off by using the keyboard shortcut Alt + Shift + D (customizable in settings).

Theme Options

DarkMode+ offers multiple theme options to suit your preferences:

Default Theme

A classic dark mode with intelligent color inversion that preserves images and media. Features dark gray backgrounds and light text for optimal readability.

Sepia Theme

A warm, amber-colored dark mode that further reduces blue light emission. This theme gives websites a pleasant, vintage book appearance that's particularly easy on the eyes in low-light environments.

Midnight Theme

A sleek, modern dark theme with deep blue-black backgrounds and cool-toned accents. Perfect for night-time browsing with minimal eye strain.

Custom Theme

Create your own perfect dark mode by selecting custom colors for backgrounds, text, links, and accents. See the "Custom Themes" section for detailed instructions.

Selecting a Theme

To change your theme:

  1. Click the DarkMode+ icon in your browser toolbar
  2. Select your preferred theme from the dropdown menu
  3. The change will apply immediately

Custom Themes

One of the most powerful features of DarkMode+ is the ability to create your own custom theme with personalized colors.

Creating a Custom Theme

  1. Click the DarkMode+ icon in your toolbar
  2. Select "Custom" from the theme dropdown
  3. Click "Customize Colors" to open the color selection panel
  4. Adjust the following colors using the color pickers or by entering hex values:
    • Background Color: The main dark background color for websites
    • Text Color: The primary color for text content
    • Links Color: The color used for hyperlinks
    • Accent Color: The color used for UI elements like buttons and highlights
  5. Preview your theme in real-time as you make adjustments
  6. Click "Save" to apply your custom theme
Recommended

Visit our Theme Suggestions page for pre-selected color palettes that work well together and provide optimal readability.

Tips for Creating Effective Custom Themes

  • Maintain sufficient contrast between background and text colors for readability
  • Choose link colors that stand out from your regular text
  • Consider using slightly muted colors to reduce eye strain
  • Test your theme on various websites to ensure it works well across different designs

Site Exclusions

There may be websites where you prefer to keep the original light mode design. DarkMode+ makes it easy to exclude specific sites from dark mode application.

Adding a Site to Exclusions

You can add a site to your exclusion list in two ways:

Method 1: From the Popup Menu

  1. Navigate to the website you want to exclude
  2. Click the DarkMode+ icon in your browser toolbar
  3. Click the "Add to Exclusion List" button
  4. The website will immediately return to its original light mode

Method 2: From the Options Page

  1. Access the DarkMode+ options page
  2. Scroll down to the "Site Exclusions" section
  3. Enter the domain name in the text field (e.g., "example.com")
  4. Click "Add" to add it to your exclusion list

Managing Exclusions

To view and manage your excluded sites:

  1. Access the DarkMode+ options page
  2. Scroll to the "Site Exclusions" section
  3. You'll see a list of all excluded domains
  4. To remove a site from the list, click the "X" button next to it
Note

Site exclusions are based on domain names. For example, excluding "example.com" will also exclude all subdomains like "blog.example.com".

Keyboard Shortcuts

DarkMode+ supports keyboard shortcuts for faster access to common functions:

Function Default Shortcut
Toggle Dark Mode On/Off Alt + Shift + D
Open Options Page Alt + Shift + O
Cycle Through Themes Alt + Shift + T

Customizing Keyboard Shortcuts

You can customize these keyboard shortcuts to your preference:

  1. Navigate to chrome://extensions/shortcuts in your browser
  2. Find DarkMode+ in the list
  3. Click the pencil icon next to the shortcut you want to change
  4. Press your desired key combination
  5. Click outside the input box to save

Settings Sync

DarkMode+ automatically syncs your settings across all your devices where you use Chrome with the same Google account.

What Gets Synced

  • Theme preferences (Default, Sepia, Midnight, or Custom)
  • Custom theme color settings
  • Site exclusion list
  • General extension settings

Requirements for Sync

To enable settings sync:

  1. Sign in to Chrome with your Google account on all your devices
  2. Ensure Chrome Sync is enabled in Chrome settings
  3. Make sure "Extensions" or "Extensions Data" is included in your sync settings
Note

There may be a brief delay before changes sync across all your devices. If you don't see your settings updated immediately, wait a few minutes or manually refresh.

Troubleshooting

If you encounter any issues with DarkMode+, here are some common solutions:

Dark Mode Not Working on Specific Sites

  • Check if the site is in your exclusion list
  • Some websites with complex designs or those using special technologies may not be fully compatible
  • Try using a different theme option as some themes work better on certain sites
  • Check if the website has its own dark mode option that might be conflicting with DarkMode+

Images or Videos Look Inverted

  • DarkMode+ tries to preserve media elements, but some may be affected
  • Try toggling dark mode off temporarily when viewing specific media
  • Right-click on the inverted image and select "Ignore in Dark Mode" (Note: this setting doesn't persist across sessions)

Performance Issues

  • On very complex websites, DarkMode+ may cause slight performance delays
  • Consider adding these sites to your exclusion list
  • Ensure your Chrome is updated to the latest version
  • Disable other extensions that might be conflicting

Settings Not Syncing

  • Verify you're signed in to Chrome with the same Google account
  • Check that Chrome Sync is enabled and includes extension data
  • Force a sync by clicking the sync icon in Chrome settings

Reinstalling the Extension

If you continue to experience issues, try reinstalling DarkMode+:

  1. Right-click the DarkMode+ icon in your toolbar
  2. Select "Remove from Chrome..."
  3. Confirm removal
  4. Reinstall from the Chrome Web Store

Note that reinstalling will reset your settings unless they were synced to your Google account.

Frequently Asked Questions

Will DarkMode+ work on all websites?

DarkMode+ is designed to work on virtually any website. Our smart algorithms apply dark mode styling while preserving images, videos, and other media elements. If you encounter a website where the extension doesn't work optimally, you can easily add it to your exclusion list or try a different theme.

How do I customize the colors in my dark mode?

To customize colors, select "Custom" in the theme dropdown menu in the extension popup. This will open a color customization interface where you can adjust background, text, link, and accent colors. You can use the color pickers or enter specific hex values for precise control.

Will using dark mode save battery on my device?

If your device has an OLED or AMOLED display (common in many smartphones and high-end laptops), using dark mode can significantly reduce power consumption since these displays don't use power to show black pixels. For LCD displays, the power savings are minimal, but you'll still benefit from reduced eye strain.
